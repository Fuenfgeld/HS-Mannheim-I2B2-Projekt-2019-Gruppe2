Anleitung für den Ordner testFlask:
1. Downloaden Sie bitte die Zip- Datei
2. Entpacken Sie bitte die Zip-Datei
3. Öffnen Sie  bitte den Ordner und klicken Sie auf env
4. Klicken Sie bitte auf den Ordner app
5. Führen Sie bitte die cmd Datei auf
6. Auf ihrem localhost mit Port 5000 sollte eine Dash Application laufen

Für die Ausführung wird auf jeden Fall Python und Flask benötigt. 
Python: https://www.python.org/downloads/

Für Flask sollte folgender Kommandozeilenbefehl ausreichen: pip install Flask

Falls es zu Problemen mit Dash kommen sollte, ist die Installation hier zu finden: https://dash.plot.ly/installation
